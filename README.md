# Mobile Phone Sales-Analytics-Dashboard (Interactive Dashoard Creation using Power BI)
## Project Objective
The objective of this project is to analyze mobile sales data to monitor sales performance, revenue trends, and transaction details. It helps in identifying growth patterns and supports data-driven decision-making through an interactive dashboard.
